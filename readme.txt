
JSP人力资源管理系统源码（SSH）
源码描述：
一、源码介绍
        JSP人力资源管理系统源码采用SSH，mysql实现，mysql导入sql文件，修改（WebContent\WEB-INF\applicationContext.xml）applicationContext.xml文件修改数据库配置信息。
        tomcat部署后，访问http://localhost:8080/PersonManager/
二、主要功能
        用户名：admin，密码：123 
        登录后可以进行添加员工，制定培训计划，添加应聘信息，制定奖惩信息，添加部门信息等操作。
三、注意事项
    开发环境为jdk1.8，数据库为mysql
